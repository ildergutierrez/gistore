Configuración SMTP para envío de correos con PHPMailer

📧 Cuentas configuradas

➕ Nueva cuenta SMTP

check_circle Cuenta guardada correctamente.
error Completa todos los campos.

📖 Guía rápida de configuración SMTP

1

Gmail — Activa la verificación en dos pasos

Ve a myaccount.google.com → Seguridad y activa la verificación en dos pasos. Luego genera una contraseña de aplicación para usar como clave SMTP. Host: smtp.gmail.com · Puerto: 587

2

Outlook / Hotmail

Host: smtp-mail.outlook.com · Puerto: 587 · SSL desactivado, TLS activado. Usa tu correo y contraseña habitual.

3

SendGrid (recomendado para envíos masivos)

Host: smtp.sendgrid.net · Puerto: 587 · Usuario: apikey · Contraseña: tu API Key de SendGrid. Ideal para enviar más de 300 correos/día.

4

Usa el botón "Probar conexión"

Antes de guardar, siempre prueba la conexión para confirmar que los datos son correctos y el servidor SMTP responde correctamente.
