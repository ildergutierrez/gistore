🔑 Claves API

⚠ 
✓ 
🔐 ¿Para qué sirve esto? Las claves API nunca se guardan en el código fuente. Se almacenan aquí en Firestore y los módulos que las necesitan (como Spotlight IA) las leen dinámicamente. Si cambias una clave aquí, el cambio aplica de inmediato sin tocar ningún archivo.
Campos clave: servicio = identificador único · origen_url = URL base de la API · modelos = lista de modelos (uno por línea) · clave = API key secreta.
