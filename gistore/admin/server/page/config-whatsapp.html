🔑 API WhatsApp Business

Configuración de la API oficial de Meta para envío de mensajes

info
GI Store usa la API oficial de WhatsApp Business (Meta Cloud API). Para usarla necesitas una cuenta de Meta Business, un número de WhatsApp Business verificado y un token de acceso permanente. Este módulo solo maneja el envío de mensajes.

🧪 Mensaje de prueba

📖 Cómo obtener tus credenciales

1

Crea una app en Meta Developers

Ve a developers.facebook.com → Mis apps → Crear app → Tipo: Business. Agrega el producto WhatsApp.

2

Configura tu número de WhatsApp Business

En la sección WhatsApp → Configuración de tu app, verifica tu número de teléfono. Copia el Phone Number ID y el WABA ID.

3

Genera un token permanente

Ve a Meta Business Suite → Configuración → Usuarios del sistema → Crea un usuario del sistema → Genera un token con permisos whatsapp_business_messaging.

4

Pega las credenciales aquí y prueba

Ingresa el token, Phone Number ID y WABA ID en el formulario. Usa el botón Probar conexión para verificar que todo funcione.

🔗 Endpoint utilizado

GI Store envía mensajes usando el siguiente endpoint de la Cloud API:

POST https://graph.facebook.com/v19.0/{Phone-Number-ID}/messages

Cuerpo del request (JSON):

{
  "messaging_product": "whatsapp",
  "to": "+573001234567",
  "type": "text",
  "text": {
    "body": "Hola [nombre], mensaje desde GI Store."
  }
}

info Los números destinatarios deben haber iniciado una conversación con la cuenta en las últimas 24 h, o debes usar plantillas aprobadas por Meta.
